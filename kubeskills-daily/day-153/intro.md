# RBAC Misconfigurations - Accidental Privilege Escalation

In this lab you will investigate a common RBAC misconfiguration where a ServiceAccount has been granted far more cluster access than intended. You will audit existing bindings, understand the scope of the damage, and remediate by creating properly scoped permissions. This scenario mirrors real incidents found during Kubernetes security audits.
