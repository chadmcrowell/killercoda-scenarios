# Control Plane Certificate Expiration - Cascading Failures

Certificate expiration in the Kubernetes control plane is a silent countdown that ends with a complete cluster outage if not addressed before the deadline. In this lab you will learn to inspect certificate expiration dates across all control plane components, understand which components are affected by each certificate, and perform the renewal steps needed to restore cluster health.
