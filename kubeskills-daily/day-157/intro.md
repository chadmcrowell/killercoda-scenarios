# Kubernetes API Deprecations - When Your Manifests Stop Working

In this lab you will work through the real-world process of identifying and fixing Kubernetes API version deprecations that have broken a set of application manifests after a cluster upgrade. You will learn how to find removed API versions, understand the migration path to their replacements, and update manifests without losing any configuration intent. This is a common CKA exam scenario.
