# Image Tag Immutability and Pull Policy Misconfiguration

Welcome to the Image Tag Immutability lab. Several deployments in this cluster are using the latest tag and operators have been reporting that rollbacks are not working as expected and different pods seem to be behaving differently. Your job is to understand why mutable tags cause these problems and then migrate the deployments to an immutable tagging strategy.
