# Priority Class Preemption Stealing Production Resources

Welcome to this lab on priority class misconfigurations. Kubernetes uses priority classes to decide which pods get resources first and which get evicted under pressure. When batch jobs carry higher priority than production services, your critical workloads become vulnerable to unexpected eviction. In this lab you will find the misconfigured priority assignments and fix them before the next pressure event.
