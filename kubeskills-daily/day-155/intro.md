# Node Pressure Eviction - Protecting the Kubelet

In this lab you will observe and understand kubelet-driven pod eviction triggered by node memory pressure. You will learn the eviction priority ordering based on QoS classes, inspect node conditions and eviction events, and modify pod resource declarations to change their eviction priority. This knowledge is essential for operating stable Kubernetes clusters under real-world resource pressure.
