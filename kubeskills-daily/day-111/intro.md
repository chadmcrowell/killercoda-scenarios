# Node Affinity Scheduling Failures

Welcome to the Node Affinity debugging lab. In this scenario your team has deployed several applications with node affinity rules but none of the pods are running. Your job is to figure out why the scheduler is refusing to place these pods and fix the configuration so the workloads can start. Understanding how affinity rules interact with node labels is a critical skill for any Kubernetes administrator.
