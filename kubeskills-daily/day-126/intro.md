# Node Disk Pressure - Diagnosing and Resolving Eviction Events

In this lab you are responding to an alert that a node has entered DiskPressure state and multiple pods have been evicted. A logging workload has gone rogue and filled the node disk. Your job is to identify the problem using node conditions and events, free up disk space by addressing the runaway workload, and then configure proper resource limits so this cannot happen again.
