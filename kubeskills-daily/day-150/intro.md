# Pod IP Address Exhaustion - Diagnosing and Recovering Cluster Network Capacity

Welcome to the Pod IP Exhaustion lab. New pods are stuck in Pending and the scheduler events are pointing to a network allocation failure rather than resource pressure. Your job is to diagnose why IP addresses are exhausted, identify pods or allocations that should have been cleaned up, and restore the cluster to a state where new pods can be scheduled.
