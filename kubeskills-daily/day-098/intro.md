# Pod Disruption Budget Rolling Update Stall

Welcome to this lab on Pod Disruption Budget misconfigurations that block rolling updates. You will investigate a stalled deployment rollout, understand how PDB settings interact with rolling update strategies, and fix the configuration to allow updates to complete. This scenario reflects a common production situation where safety mechanisms become obstacles.
