# ConfigMap Hot Reload and Stale Configuration Debugging

Welcome to this ConfigMap configuration management lab. An application team has updated a ConfigMap with new database connection settings but the running application is still connecting to the old database endpoint. They updated the ConfigMap over an hour ago and cannot understand why the change has not taken effect. Your job is to explain the propagation behavior, demonstrate the difference between the two consumption methods, and implement a reliable solution for configuration updates going forward.
